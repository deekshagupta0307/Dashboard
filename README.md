# Employee Management Dashboard

## Project Overview
This project is a React.js based Employee Management Dashboard built as part of a hiring assessment.
The application allows authenticated users to manage employee records with full CRUD functionality,
search and filter employees, upload profile images, and print employee data.

## Features
- Mock Authentication with Protected Routes
- Dashboard Summary (Total, Active & Inactive Employees)
- Add, Edit, Delete Employees
- Profile Image Upload with Preview
- Active / Inactive Status Toggle
- Search Employees by Name
- Filter Employees by Gender and Status (Combined Filters)
- Print Employee List
- Clean, Modern & Responsive UI
- Data Persistence using Local Storage

## Tech Stack
- React.js
- Vite
- React Router DOM
- Context API
- Tailwind CSS
- Local Storage
